# Habitto

> Минималистичный трекер привычек — одним HTML-файлом, без зависимостей, с хранением данных в браузере.

---

## Быстрый старт

Скачайте `habitto.html` и откройте в браузере. Всё.

```
habitto.html   ← единственный файл
```

Никаких сборщиков, серверов и установки пакетов. Работает офлайн.

---

## Функциональность

### Привычки
- Создание с именем, иконкой, цветом акцента и заметкой
- Редактирование любого поля через модальное окно
- Удаление с подтверждением через toast-уведомление
- Отметка выполнения одним кликом с анимацией конфетти

### Частота выполнения
Три режима на выбор при создании и редактировании привычки:

| Режим | Описание |
|---|---|
| **Каждый день** | Привычка активна все 7 дней |
| **Конкретные дни** | Визуальный пикер Пн–Вс, любая комбинация |
| **N раз в неделю** | Счётчик от 1 до 7, дни на своё усмотрение |

### Категории
- 7 встроенных: Здоровье, Спорт, Обучение, Работа, Личное, Питание, Сон
- Создание своих категорий прямо в форме
- Удаление кастомных категорий
- Фильтрация списка привычек по категории

### Статистика и визуализация
- Счётчик выполнения за сегодня (сделано / запланировано)
- Лучший стрик среди всех привычек
- Процент выполнения за текущую неделю
- Прогресс-бар на день с анимацией
- Недельный обзор с цветными точками по каждой привычке
- 21-дневный мини-календарь на каждой карточке (раскрывается по кнопке)
- Серия дней подряд (стрик) на каждой привычке

### Интерфейс
- Тёмная и светлая тема с переключением
- Сортировка: по дате добавления, имени, стрику, категории
- Hover-эффекты на всех интерактивных элементах
- Toast-уведомления о действиях
- Shake-анимация при попытке сохранить пустую форму
- Адаптивная вёрстка для мобильных

## Хранение данных

Все данные хранятся в `localStorage` браузера, ключи:

| Ключ | Содержимое |
|---|---|
| `hb3_h` | Массив привычек (JSON) |
| `hb3_l` | Логи выполнения — `{ habitId: ["2025-01-01", ...] }` |
| `hb3_cats` | Кастомные категории (JSON) |
| `hb3_theme` | Текущая тема (`"dark"` / `"light"`) |

Данные не покидают устройство. Очищаются вместе с данными браузера.

## Структура привычки

```js
{
  id:      "1718000000000",   // timestamp
  name:    "Утренняя пробежка",
  cat:     "спорт",
  freq: {
    type:  "days",            // "daily" | "days" | "ntimes"
    days:  [0, 2, 4],         // индексы: 0=Пн, 1=Вт, ..., 6=Вс
    n:     3                  // используется при type="ntimes"
  },
  color:   "#00cba0",
  icon:    "run",
  note:    "30 мин на свежем воздухе",
  created: 1718000000000
}
```

## Браузерная поддержка

Работает в любом современном браузере: Chrome, Firefox, Safari, Edge.  
Требует `color-mix()` — поддерживается с Chrome 111, Firefox 113, Safari 16.2.
